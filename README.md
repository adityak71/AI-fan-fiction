## 📌 Overview

**AI FanFic** is a modern, interactive frontend application that allows users to generate immersive fan fiction stories using AI. It supports multiple creative modes including **Text Chat**, **Voice Chat**, and **Image Generation**, enabling users to bring their imagination to life.

> ⚡ Built as a frontend-only project (no backend required), leveraging AI APIs for real-time content generation.

---

## 🎯 Features

* 💬 **Text Story Chat**

  * Generate stories in real-time using prompts
  * Streaming AI responses for immersive storytelling

* 🎤 **Voice Story Chat**

  * Speak your ideas and let AI convert them into stories
  * Works best on modern browsers like Chrome & Edge

* 🎨 **Image Generation**

  * Convert fan fiction scenes into visuals
  * Multiple art styles: Cinematic, Anime, Watercolor, etc.

* 🌌 **Fandom-Based Story Settings**

  * Choose from popular universes:

    * Harry Potter
    * Marvel
    * Star Wars
    * Naruto
    * Game of Thrones
    * And more...

* 🎭 **Genre & Tone Customization**

  * Genres: Adventure, Romance, Mystery, Comedy, etc.
  * Tones: Epic, Dark, Humorous, Whimsical, etc.

* 🌗 **Light & Dark Theme Support**

  * Fully responsive UI with smooth theme switching

* 🔐 **Authentication UI (Frontend Only)**

  * Login/Logout UI with user display

## 🛠️ Tech Stack

* ⚛️ React.js (Frontend Framework)
* 🎨 Tailwind CSS / Custom CSS
* 🧠 Google Gemini API (for AI generation)
* 🎤 Web Speech API (Voice Input)
* 🎭 Framer Motion (Animations)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/adityak71/AI-fan-fiction.git
cd AI-fan-fiction
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file in the root directory:



```
copy .env.example content to .env and replace the '*****' with your API keys.

for firebase api keys you can directly paste your keys on \AI Fan Fiction\src\firebase.js file.

```

---

### 4️⃣ Run the Development Server

```bash
npm run dev
```

---

### 5️⃣ Open in Browser

```
http://localhost:5173
```

---

## 🧠 How It Works

1. User selects **Fandom + Genre + Tone**
2. Chooses a mode:

   * Text → Input prompt
   * Voice → Speak idea
   * Image → Describe scene
3. AI processes input via API
4. Response is streamed/rendered dynamically
